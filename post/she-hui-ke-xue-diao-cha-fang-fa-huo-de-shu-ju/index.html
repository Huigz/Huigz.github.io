[社会科学调查方法] 获得数据 📃

2020-05-22

Data generation

  • primary data
  • secondary data

primary data : 研究人员通过 survey, interview ... 获取的 first-hand sources.

secondary data : 从其他人/机构获取的数据 eg : government statistics

Type of survey

What is survey?

Survey is the process of collecting data through a questionnaire that asks individuals the same questions

Type of information collected

  • Descriptive 描述性信息(社会经济特征/教育水平/关于调查对象的特征)
  • Behavioral 行为模式(吃不健康的食物/步行或骑车去学校)
  • Attitudina 态度(对公共事件的看法/满意程度)

Type of survey

  • face to face
  • telephone
  • Email/online

如果调查对象没有文化 face to face 是必要的

Survey process

  • Who?
  • When?
  • Where?
  • How?
  • What?

  1. Operationalize your concepts
    运用你的概念

  2. Formulate your questions
    规划你的问题

  3. Decide on response categories
    确定回应类别

  4. Provide background information and clear instructions
    提供背景信息和清晰的说明

  5. Decide organization and length
    决定组织和长度

  6. Create aesthetically pleasing layout and design
    创造美观的布局和设计

1. Operationalize concepts 概念运用

Turning abstract concepts into measurable variables

将抽象概念转化为可测量的变量

eg :

腐败 : 主观概念 -> 寻找定义 -> 通过媒体曝光的政客非法活动的事件/报道 数量 -> 客观统计

2. Formulate your questions 明确问题

避免 复杂 / 双管齐下 / 模糊 / 引导性问题

避免 攻击性/敏感性问题

3. Decide on response categories 确定回应类别

  • Closed responses 封闭式回应
  • Open responses 开放式回应

Closed responses eg : Likert scale

Open responses : answer using own words

likert scale 李克特量表

  • 强烈反对
  • 反对
  • 中立
  • 赞同
  • 强烈赞同

4. Provide background information and clear instructions 提供背景信息和清晰的说明

调查页头标明

调查目的 / 工作单位&联系方式 / 同意书 / 使用说明等

5. Decide organization and length 决定组织和长度

length

不宜超过20分钟 不宜太长/短

organization

不以威胁/困难/尴尬/敏感等复杂问题作为开头

eg : 以收入问题作为调查开头

问题顺序的影响:

Primacy Effect : 答题者倾向第一选择 (paple / online survey)

Recency Effect : 答题者倾向于听到的最近的选择 (face to face / telephone survey)

6. Create aesthetically pleasing layout and desig 创造美观的布局和设计

美观 简洁 专业

阅读

心理学归因理论 —协方差模型

哈罗德·凯利（Harold Kelley）的协变量模型是一种归因理论，人们在其中进行因果推断，以解释为什么其他人和我们自己以某种方式行事。它与社会感知和自我感知有关（Kelley，1973）。
