心理学归因理论 —协方差模型

心理学 协方差理论
2020-05-31

哈罗德·凯利（Harold Kelley）的协变量模型是一种归因理论，人们在其中进行因果推断，以解释为什么其他人和我们自己以某种方式行事。它与社会感知和自我感知有关（Kelley，1973）。

归因理论 ——协方差模型

协方差模型 Covariation model

—— 哈罗德·凯利（Harold Kelley）

归因标准:

  • 共识
  • 独特性
  • 一致性

归因结论:

  • 内部个人
  • 外部刺激

共识

指不同人行为的共同变化

共识高 -> 归因为外部刺激
共识低 -> 归因为内部个人

eg.

许多人认为lisa漂亮 -> 人们归因为外部刺激(lisa)

只有Tom认为lisa漂亮 -> 人们归因为内部个人(Tom)

独特性

情况变化下人行为与平时行为差异程度
(一般行为还是仅针对某特情况的行为)

情形下人行为差异小 -> 独特性低 -> 归因为内部个人
清形下人行为差异大 -> 独特性高 -> 归因为外界刺激

eg.

lisa不喜欢所有浪漫电影 -> lisa喜欢泰坦尼克号 -> 独特性高 -> 归因为外部因素(泰坦尼克号)

lisa不喜欢所有浪漫电影 -> lisa不喜欢泰坦尼克号 -> 独特性低 -> 归因为lisa个人

一致性

随时间的协变 行为的一致程度

eg.

lisa一直很谦虚 -> 一致性高 -> 归因为个人(lisa很慷慨)

lisa只有在面试的时候很谦虚 平时不谦虚 -> 一致性低 -> 归因为外部刺激(面试使lisa谦虚)

归因模型

  • 低共识，低独特性，高一致性=个人归因
  • 高共识，高独特性，高一致性=激励归因
  • 高共识，低独特性，低一致性=情境归因
