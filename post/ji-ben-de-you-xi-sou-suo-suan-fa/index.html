基本的游戏搜索算法

2020-06-04 2 min read # 蒙特卡洛树 # AI人工智能 # Mini-Max算法 # Alpha-beta算法

Search in Game

Game state tree

双方对战游戏中，模拟自我和对手可能的游戏状态

应用场景

  • 自我顺序选择的最有利选择的状态是对手最不利状态
  • 分析越多的状态越有利

Mini-Max 算法

术语：

  • Max node: 最大值节点 指自我顺序最优选择
  • Min node: 最小值节点 指对方最优选择(相对自我最坏选择)

方法:

从端末节点计算分数，向上层节点递推，得出顶端节点可选择的最优值.

缺陷：

每种状态下有n种选择，x步后，将出现n的x次方种选择。计算量过于庞大.

Alpha-Beta 算法

Mini-Max算法的改良 又称剪枝算法 即排除不必要的节点

Mini-Max算法向上递推的过程中，根据对方和自我的最坏最优解逻辑，当子节点分数给出时，可推断出父节点的规则.

方法:

  • alpha cut-off: Min-node状态下，当前节点值<=父节点时 当前子节点探索中断
  • beta cut-off: Max-node状态下，当前节点值>=父节点时 当前子节点探索中断

MCTS 蒙特卡洛树算法

UCT：选择展开和回溯的决策算法

Eg

过程：

  1. 选择
  2. 展开（确定模拟节点）
  3. 模拟（评价）
  4. 回溯

解释：

  1. 选择UCT值最高的节点，为展开节点的UCT值为无限大

  2. 在被选择的节点种，选择一个未被访问过的子节点，展开

  3. 从新展开的节点开始，随机游戏（模拟），直到游戏结束，得到游戏结果

  4. 将随机模拟的游戏结果回溯给父节点，重新计算所有的UCT值

根据探索时间的限制 算法停止探索时 最后根据UCT的值作出最优选择
